### 웹사이트 이용 테스트  
#### 0. 접속  
* 크롬 브라우저에서 https://client.nudo-study.cf/ 접속  
#### 1. 로그인  
* 우측 상단의 메뉴 ⇒ 로그인으로 아이디 aaaa, 비밀번호 1234로 로그인합니다.  
<img src="https://user-images.githubusercontent.com/76865900/172979868-5e60b327-c664-4c82-9157-c27cffcb05a7.png" width="700"> 

#### 2. 캠스터디 진행 및 공부 시간 자동 측정 기능 확인  
* 우측 상단의 메뉴 ⇒ "참여 중인 스터디" 페이지로 이동합니다.  
* [공부 시작] 버튼 클릭 시 캠스터디 공간으로 입장합니다.  
<img src="https://user-images.githubusercontent.com/76865900/172980624-f2804827-8632-4921-9a77-3f24bc4e5dc1.png" width="700">  

* 좌측 하단의 mode 버튼을 클릭하여 on 상태로 변경하면 공부 시간 자동 측정 모드로 진입합니다. (on: 자동 측정 모드, off: 수동 측정 모드)  
* 화면에 **펜 잡은 손**이 나오면 스탑워치에 공부 시간이 누적됩니다. 
* 좌측 상단의 [나가기] 버튼을 클릭하여 캠스터디를 종료할 수 있습니다.  
<img src="https://user-images.githubusercontent.com/76865900/172981694-a2c4e0e8-519e-4e05-bc64-a3192e1f968d.png">   

#### 3. 스터디 관리 : 출석 체크, 벌금 정산, 공부 시간 확인  
* 참여한 스터디의 [관리] 버튼을 클릭하여 출석체크/벌금정산/공부시간 현황을 확인할 수 있습니다.  
<img src="https://user-images.githubusercontent.com/76865900/172982903-327c44eb-1e66-4622-ab47-3828a0896688.png" height="500">  

* 공부시간 탭 클릭 후 빈 화면에서 날짜를 클릭해야 해당 날짜의 공부 시간 기록이 확인됩니다.  
<img src="https://user-images.githubusercontent.com/76865900/172983585-f836687b-b30d-481f-b9b9-44a2a49af257.png" height="250">  

#### 4. 스터디 모집  
* 우측 상단의 메뉴 ⇒ "스터디 목록" 페이지로 이동합니다.  
* [JOIN] 버튼을 클릭해 모집이 완료되지 않은 스터디 중 원하는 스터디에 가입할 수 있습니다.  
* [+] 버튼을 클릭하면 직접 스터디를 모집할 수 있습니다.  
<img src="https://user-images.githubusercontent.com/76865900/172984203-b9d81f04-7e53-4f8c-a0df-8fe67ee7105d.png" height="500">  

#### 5. 스터디 랭킹  
* 우측 상단의 메뉴 ⇒ "스터디 랭킹" 페이지로 이동하면 전날의 스터디별 공부 시간 랭킹을 확인할 수 있습니다.  
<img src="https://user-images.githubusercontent.com/76865900/172985028-f5a51ec2-9aaa-48bb-b5a7-6b514f7f7e4a.png" height="500">
